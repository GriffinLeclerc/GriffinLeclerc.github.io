Weather Sender

Too long had I been burdened by inaccurate or bloated weather services.
So I devised a plan, a plan in which I would send myself the weather.

Thanks to the wonderful website wttr.in,
I was able to gather weather data in json format: wttr.in/?format=j1
And it was simple once I located the information I was interested in.

Thanks to my phone provider for allowing me to sent texts over SMTP,
I am able to notify myself of the weather each morning.

In addition to learning more about json, I learned about the use of configuration
files and bash scripting.

I decided it best to use a configuration file for users to specify their location
and recipients, rather than having them edit the script directly.
I then wrote bash script to first generate the configuration file, request (and wait for)
the user fill relevant information, then run the script.

Finally, in order to protect the user's privacy, the script requires the user to
set two system environment variables to their email address and password.
This discourages users from storing their sensitive information in plain text.

The source code can be found here.

Eagle Scout Service Project

In early October 2014, I was a Life Scout in Boy Scout Troop 292.
A few of my coullegues were underway with their Eagle Scout service projects.
These projects included construction of an ice skating hut outide a local pond,
and refurbishment of a popular walking path in the woods.

I wanted a less traditional project that would immediately and directly effect others.

After a bit of research, I came into contact with a representative of the Computer Technology Assistance Corperation.
They allowed me to use their space over three working days to store, assess, and refurbish
the 147 computers I had collected from those who had no use for their old systems.

Of the 147, 82 were fully functional and ready for distribution, 37 were disassembled for
functional components, and 28 were disassembled for raw materials.
